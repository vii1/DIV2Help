Declaración incorrecta de puntero a una estructura

El compilador esperaba, en la posición indicada del programa, el nombre de una estructura declarada anteriormente, para poder declarar un puntero a la misma.

Para definir un puntero a una estructura de datos se debe utilizar la siguiente sintaxis:

STRUCT POINTER nombre_estructura nombre_puntero;

Siendo nombre_estructura el nombre de una estructura de datos declarada en una posición anterior del programa, y nombre_puntero nombre del puntero la estructura que se está declarando.
