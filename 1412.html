c_m8

Constante: c_m8

Esta constante se utiliza para asignarla a la variable local predefinida ctype que sirve para definir el tipo de coordenadas que tendrá un proceso. Su valor es 3.

Éste es el valor asignado a ctype para que las coordenadas del gráfico del proceso se interpreten como referidas a una ventana de modo 8, a coordenadas sobre el mapa de sectores tridimensional que se muestra en la misma.

Para más información sobre las ventanas de modo 8, se puede acceder a la función start_mode8(), utilizada para activarlas.

Existen otras constantes utilizadas para otros sistemas de coordenadas, la lista completa es la siguiente:

    c_screen - Coordenadas de pantalla
    c_scroll - Coordenadas de scroll
    c_m7 - Coordenadas de modo 7
    c_m8 - Coordenadas de modo 8
