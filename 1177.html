min_int

Constante: min_int

Esta constante define el valor mínimo que puede almacenar cualquier dato en este lenguaje. Este valor es -2147483648.

Todos los datos son enteros con signo de 32 bits en este lenguaje. Por ello, únicamente se pueden manejar números enteros dentro del rango ( -2147483648 ... +2147483647 ).

Cuando el resultado de una operación aritmética exceda dicho rango, el sistema no advertirá de ningún error, por lo que se deben extremar las precauciones para que ésto no suceda.
