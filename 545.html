Se esperaba otro parámetro

En las llamadas a procesos o funciones se deben poner tantos valores o expresiones como párametros se requieran, todos dentro del mismo paréntesis y separados por comas (,). Este error se produce cuando se cierra el paréntesis tras una coma, como en el siguiente ejemplo:

Ejemplo: 
PROGRAM mi_juego;
BEGIN
    signal(id, ); // Error: Se esperaba otro parámetro.
END


Puede haber pasado dos cosas; se ha puesto una coma accidentalmente tras el último parámetro, o bien se ha omitido el último parámetro (como en el ejemplo anterior).
