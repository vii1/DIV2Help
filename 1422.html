LOCAL m8_wall

m8_wall; // Número de pared del mapa de sectores

Esta variable predefinida es LOCAL, lo que significa que cada proceso tendrá su propio valor en su variable m8_wall.

Esta variable local se utiliza exclusivamente en los procesos de un modo 8 (aquellos que tienen asignado c_m8 a ctype).

En m8_wall el sistema guardará, de forma automática, el número de pared contra la que ha colisionado el proceso.

Esta información puede ser útil para detectar distintas zonas activas dentro de un mapa de sectores (como puertas, interruptores, etc.).

Nota: Para más información sobre el modo 8, véase la estructura global m8 y la función start_mode8(), que es la empleada para inicializar regiones de este tipo.
