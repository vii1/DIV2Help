Datos de tipo INT

Datos numéricos enteras entre -2147483648 y +2147483647 (valores definidos en las constantes min_int ... max_int).

Este es el dato de tipo básico del lenguaje, cuaundo no se especifique el tipo de un dato, este será INT.
