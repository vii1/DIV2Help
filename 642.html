Esperando un nombre

Para definir una tabla de bytes se debe especificar el nombre de la misma tras la palabra reservada BYTE.

Ejemplo: 
PROGRAM mi_juego;
GLOBAL
    BYTE mi_byte;
    BYTE mi_tabla1[127];
    BYTE mi_tabla2[255,255];
BEGIN
    //...
END


En este ejemplo se han definido tres datos de este tipo. El primero con el nombre mi_byte podrá contener un único valor (entre 0 y 255), el segundo con el nombre mi_tabla1 podrá contener hasta 128 de estos valores (desde mi_tabla1[0] hasta mi_tabla1[127]), y el tercero, que será una tabla de bytes de dos dimensiones, pudiendo contener un total de 65536 valores (ya que este es el resultado de multiplicar 256 por 256).
