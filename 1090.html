ELSE

Esta palabra forma parte de la sentencia IF, y sirve para especificar dónde comienza el bloque de sentencias que se debe ejecutar cuando la condición especificada en la sentencia IF no se cumpla.

El formato de la sentencia IF es el siguiente:

IF (<condición>)
    // Código a ejecutar cuando se cumpla la condición
ELSE
    // Código a ejecutar cuando no se cumpla la condición
END

El significado semántico de ELSE es "en el caso contrario ...".

La sección ELSE es opcional, pudiendo formularse la sentencia IF de la siguiente forma:

IF (<condición>)
    // Código a ejecutar cuando se cumpla la condición
END

Cuando no se requiera que se ejecute ninguna sentencia en el caso contrario.
