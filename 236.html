mkdir()

mkdir(<directorio>)

Descripción:

Crea un directorio nuevo (dentro del directorio o carpeta actual).

Se debe indicar el nombre del directorio a crear como un literal (un texto entre comillas).

La forma de utilización y posibilidades de esta función son las mismas que las del comando MD de la línea de comandos del sistema operativo.

Todos los programas se sitúan inicialmente en el directorio en el que se encuentra su ejecutable principal.

Se pueden cambiar el directorio en el que está situado un programa (en el que realizará sus operaciones de archivos) con la función chdir(), y con la función remove() se pueden eliminar los directorios existentes.

Programa ejemplo: 
PROGRAM ejemplo_mkdir;
BEGIN
    mkdir("temporal"); // Crea un nuevo directorio

    // ...
    remove("temporal");
END


Este pequeño ejemplo muestra como debe utilizarse la función mkdir() para crear un directorio, y después lo elimina con la función remove().

La función setdrive() permite establecer la unidad de disco actual y la función getdrive() averiguarlo; todos los accesos a disco (lectura y escritura de archivos) se efectúan siempre dentro de una unidad seleccionada que, por defecto, será la unidad en la que se ejecute el programa.

Nota: Se puede obtener la lista de archivos de un directorio con la función get_dirinfo().
