Excedida la capacidad del vector de nombres

Se excedió el buffer para almacenar los nombres del programa; este error puede deberse únicamente a que se han utilizado nombres de datos y procesos demasiado largos.

La única solución para poder compilar el programa es sustituir algunos de estos nombres por otros más pequeños.
