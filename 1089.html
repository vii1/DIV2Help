DUP

Esta palabra sirve para definir listas de constantes (valores numéricos), a base de repetir un patrón sencillo. Se utiliza para inicializar los valores iniciales de tablas y estructuras.

Su sintaxis es la siguiente:

<número de veces> DUP (<secuencia a repetir>)

Es equivalente a poner la <secuencia a repetir> el <número de veces> que se indica.

Por ejemplo, la secuencia 1, 2, 3, 1, 2, 3, 1, 2, 3 sería equivalente a la secuencia 3 DUP (1, 2, 3).
