GLOBAL fps

fps; // Número de imágenes por segundo

La variable global predefinida fps (frames per second) mantiene el número de imágenes por segundo que está mostrando un programa.

Esta variable es actualizada por el gestor de procesos de DIV Games Studio automáticamente. No se mantiene el valor instantáneo, sino una media de los últimos segundos, por ello se necesita cierto tiempo de ejecución para que el valor de fps se estabilice.

La función set_fps() permite establecer el número de imágenes por segundo del programa, pero de esta forma se establece el número de imágenes ideal, y no el real (que es el que contabiliza fps).

Es decir, que esta variable sirve de cuenta para saber cuantas imágenes se están volcando a vídeo por cada segundo, como media.

Nota: La variable fps mantiene un dato de sólo lectura, no se conseguirá ningún efecto modificando su contenido desde un programa.
