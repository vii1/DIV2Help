Llegó el final dentro de un comentario

Ha llegado el final del programa dentro de un comentario de varias líneas (los definidos entre los símbolos /* y */).

Este error se debe a que algún comentario no se ha cerrado, por ello se debe revisar que todos los comentarios abiertos con el símbolo /* tengan su correspondiente símbolo */ cerrándolos.

Ejemplo: 
PROGRAM mi_juego;
BEGIN
    /*Inicio del comentario
    FRAME;
END // Error: Llegó el final dentro de un comentario.
