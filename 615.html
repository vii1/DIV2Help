El nombre no es nuevo

Se ha utilizado un nombre de programa no válido pues éste ya estaba predefinido como nombre de otro objeto. Debe sustituir dicho nombre por otro similar.

Para saber a qué objeto está identificado con dicho nombre puede acceder al glosario.

Ejemplo: 
PROGRAM x; // Error: El nombre no es nuevo.
BEGIN
    //...
END


En este ejemplo se pretendía utilizar como nombre del programa x, que está predefinido como nombre de la variable local que indica la coordenada x de los procesos.
