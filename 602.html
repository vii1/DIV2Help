Excedida la capacidad de la tabla de objetos

Se excedió el espacio disponible para guardar registros de los datos y los procesos del programa.

Este error no debería suceder nunca pero si ha sucedido tiene difícil solución; el programa es demasiado grande para esta versión del compilador.

Si existe una nueva versión del compilador o algún 'patch' que actualice esta limitación, debe actualizarse.

En caso contrario la única opción es eliminar del programa todos los procesos y datos que ya no se requieran y/o no sean imprescindibles.
