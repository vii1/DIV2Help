Se esperaba una coma

Se está accediendo una tabla indicando menos dimensiones de las requeridas.

Es decir, que una tabla de dos dimensiones se prentende utilizar como si fuera de sólo una dimension, o bien que una tabla de tres dimensiones se prentende utilizar como si fuera de dos.

Ejemplo: 
PROGRAM mi_juego;
GLOBAL
    tabla1[9,9];
BEGIN
    //...
    tabla1[0]=0; // Error: Se esperaba una coma.
END


Revise la declaración anterior en el programa de dicha tabla de datos.
