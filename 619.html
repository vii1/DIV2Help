Esperando [=]

Para definir una constante se debe poner un símbolo = tras el nombre de la constante.

El compilador ha entendido que la última palabra definida era el nombre de una nueva constante y, en la posición que se indica, se esperaba que se mostrara el símbolo de asignación (=).

Ejemplo: 
PROGRAM mi_juego;
CONST
    a=0;
    b=1;
    c,d; // Error: Esperando [=].
BEGIN
    //...
END
