Datos de tipo BYTE

Datos numéricos enteros entre 0 y 255 (enteros de 8 bit sin signo).

Las tablas de tipo BYTE pueden significar un importante ahorro de memoria, respecto a las tablas de tipo INT (ocupan 4 veces menos memoria). Pero serán más rápidas las operaciones con datos de tipo INT.

Ver la alineación de datos en memoria, donde se muestra la ocupación de memoria de los diferentes tipos de datos.
