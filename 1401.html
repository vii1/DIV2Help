Tipos de datos

Al margen de los datos ya conocidos de la primera versión (que ahora se conocen como INT), existen datos de los tipos STRING (cadenas de texto), WORD (enteros positivos de 16 bit), BYTE (enteros positivos de 8 bit) y POINTER (punteros, de todos estos tipos). Además se pueden definir tablas y estructuras (también de cualquiera de estos tipos) de 2 o 3 dimensiones (ver la declaración de una tabla).

INT - Datos numéricos enteras entre -2147483648 y +2147483647 (min_int ... max_int).

WORD - Datos numéricos enteros entre 0 y 65535 (enteros de 16 bit sin signo).

BYTE - Datos numéricos enteros entre 0 y 255 (enteros de 8 bit sin signo).

POINTER - Punteros a datos de cualquier otro tipo.

STRING - Cadenas de texto (tablas de carácteres de 8 bit sin signo, entre 0 y 255)

Nota: La palabra reservada INT puede omitirse, ya que todos los datos serán de este tipo por defecto (enteros de 32 bit con signo).
