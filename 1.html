Cómo usar el sistema de ayuda

Las páginas de ayuda se manejan con el ratón, aunque también se pueden utilizar los siguientes controles del teclado.

Cursores - Desplazar el texto línea a línea.
Re/Av.Pág. - Retroceso y avance de página.
ESC - Quitar la ventana de ayuda.
Borrar - Volver a la página anterior.

Estos controles se podrán utilizar únicamente cuando la ventana de ayuda esté seleccionada (para seleccionar una ventana basta con pulsar con el ratón sobre la misma).

En la parte inferior derecha aparece un botón que sirve para modificar el tamaño de la ventana de ayuda; para ello se debe mantener el ratón pulsado mientras se desplaza este verticalmente.

Los textos que aparecen en color blanco son referencias a otras páginas de ayuda (el cursor del ratón se mostrará como una pequeña mano al situarse sobre ellas). Para visualizar la página de ayuda referenciada bastará con pulsar sobre estos textos.

Los ejemplos son programas o partes de programas que aparecen en la ayuda para demostrar la forma de utilizar alguna de las capacidades del lenguaje de programación. Todos ellos comienzan con una línea en color blanco, como se muestra a continuación.

Programa ejemplo: 
PROGRAM ejemplo;

BEGIN
    // ...
END


Para extraer el ejemplo de la ventana de ayuda basta con pulsar sobre dicha línea. Estos programas pueden ser ejecutados con la tecla F10 y para finalizar su ejecución se deben pulsar las teclas ALT+X.

En la parte superior de la ventana de ayuda aparecen una serie de botones que permiten, respectivamente, visualizar las páginas anteriores (<< y >>), mostrar algunos de los índices más importantes (Indice, Glosario y Funciones), enviar la página actual a la impresora (Imprimir), y obtener ayuda sobre como usar este hipertexto (?, que accederá a esta misma página).
