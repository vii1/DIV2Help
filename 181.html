net_get_games()

net_get_games()

Retorna:

Partidas activas para este juego.

Descripción:

Devuelve el número de partidas activas para este juego (para conexiones en red local). Esta función sirve para ver si existe alguna partida activa en un momento determinado.

Para establecer una conexión se utiliza la función net_join_game(), bien sea para crear partidas o para conectarse a las mismas.

Programa ejemplo: 
PROGRAM ejemplo_net_get_games;

GLOBAL
  n;  // Partidas activas.

BEGIN

  // ...

  n=net_get_games();

  IF (n>0)
    // ...
  END

  //...

END


Unicamente es útil para conexiones IPX, careciendo de sentido en conexiones por cable serie o a través de módem (ya que en estos casos únicamente puede haber una partida).
