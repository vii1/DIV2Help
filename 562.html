Uso: IMPORT "nombre.dll"

El compilador esperaba que, tras la palabra reservada IMPORT, apareciera entre comillas el nombre del archivo.

Para declarar el uso de una librería externa se debe poner una declaración como en el siguiente ejemplo.

Ejemplo: 
PROGRAM mi_juego;
IMPORT "libreria.dll";
BEGIN
    //...
END


Este programa declara el uso de la librería denominada libreria.dll (supuesto nombre de una hipotética librería de funciones externas).

La declaración debe hacerse justo tras la cabecera del programa, es decir, tras la declaración PROGRAM nombre_de_programa;. Pudiendo hacerse varias declaraciones IMPORT seguidas, para declarar el uso de más de una librería.
