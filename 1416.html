GLOBAL channel[]

channel[31]; // Utilización de los canales de sonido

La tabla globlal channel mantiene información sobre la utilización de los diferentes canales de sonido (hasta 32) del sistema de audio.

Cada vez que se toca un sonido con la función sound(), se envía el mismo a uno de estos canales. Y cuando se inicia la reproducción de un módulo musical con la función song(), se reservan un determinado número de canales para el mismo (hasta 24).

En esta tabla, cada posición representa a uno de estos canales de sonido, y el sistema pondrá a 1 cada posición cuando se utilice su respectivo canal.

La función change_channel() permite modificar los parámetros de cada uno de estos canales.

Nota: El sistema nunca pondrá a 0 las posiciones de la tabla channel, debe ser el programa el que borre la tabla, para de esta forma poder determinar cuando se utiliza nuevamente un canal.
