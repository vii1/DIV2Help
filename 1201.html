Datos locales predefinidos

A continuación se describe cada uno de los datos locales predefinidos.

STRUCT reserved - Parámetros internos de los procesos

angle - ángulo del gráfico del proceso
bigbro - Identificador del hermano mayor del proceso
cnumber - Números de ventanas de scroll o modo 7
ctype - Tipo de coordenadas utilizadas por el proceso
father - Identificador de padre del proceso
file - Código del fichero del gráfico del proceso
flags - Indicadores de espejado y transparencia
graph - Código del gráfico asignado al proceso
height - Altura del proceso en ventanas de modo 7
m8_nextsector - Sector hacia el que se dirige el proceso
m8_sector - Número de sector en el que está el proceso
m8_step - Altura que puede subir el proceso
m8_wall - Pared del mapa contra la que se colisiona
priority - Prioridad de ejecución del proceso
radius - Radio de los procesos de modo 8
region - Región de pantalla asignada al proceso
resolution - Resolución de las coordenadas del proceso
size - Tamaño del gráfico del proceso
smallbro - Identificador del hermano pequeño del proceso
son - Identificador del hijo del proceso
xgraph - OFFSET de la tabla gráfica del proceso
x - Coordenada horizontal del proceso
y - Coordenada vertical del proceso
z - Plano de profundidad del gráfico del proceso
