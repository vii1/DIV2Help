reset_sound()

reset_sound()

Descripción:

Función avanzada, únicamente para usuarios muy experimentados. Reinicializa el sistema de sonido.

Esta función sirve para activar unos nuevos parámetros del hardware de sonido.

Se deben establecer los siguientes valores de la estructura global setup:

    setup.card
    setup.port
    setup.irq
    setup.dma
    setup.dma2

Esta función se utiliza generalmente dentro de los programas de configuración del sistema de sonido (ver setup_program).

Para activar el resto de los valores de la estructura setup, los referentes al volumen del mezclador (mixer), se debe invocar a la función set_volume(). Los valores para establecer el volumen son:

    setup.master
    setup.sound_fx
    setup.cd_audio

Ver: set_volume() - Estructura setup - setup_program
